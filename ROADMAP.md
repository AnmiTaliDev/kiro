# Kiro — Roadmap

Planned features in priority order. No fixed deadlines — releases happen when things are ready.

---

## High Priority

- [ ] **Flatpak packaging** — publish to Flathub
- [ ] **AUR package** — `kiro` and `kiro-git` for Arch Linux
- [ ] **CI/CD** — GitHub Actions: build and validate on every push
- [ ] **Search in scrollback** — `Ctrl+Shift+F`
- [ ] **Tab reordering and renaming** — drag to reorder, double-click to rename
- [ ] **Dynamic tab titles** — show current directory or running command

## Medium Priority

- [ ] **Split panes** — horizontal and vertical splits, keyboard navigation between them
- [ ] **Profiles** — named configs with their own shell, colors, font, and working directory

## Low Priority

- [ ] **SSH quick-connect** — pick a host from `~/.ssh/config` directly from the app
- [ ] **More translations** — German, French, Spanish, Chinese
